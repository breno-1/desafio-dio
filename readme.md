# Resolvendo Problemas de Python com IA 🤖

Este repositório contém resoluções de desafios de lógica em Python, desenvolvidos com o auxílio do **GitHub Copilot**.

## 🎯 Objetivos
O foco deste projeto foi explorar como ferramentas de IA podem aumentar a produtividade e sugerir soluções eficientes para problemas comuns de programação.

## 🛠️ Desafios Resolvidos
1. **Manipulação de Strings**: União de informações de entrada do usuário.
2. **Operações Matemáticas**: Cálculos básicos com entradas numéricas.
3. **Repetição de Dados**: Uso de estruturas para replicar informações.

## 🧠 Aprendizados
Através do uso do Copilot, pude observar:
- Sugestões de sintaxe mais rápidas.
- Autocompletar inteligente para funções nativas do Python.
- Melhores práticas de nomenclatura de variáveis.


## 🌍 Contribuindo para Projetos Open Source
O GitHub é o coração da cultura de código aberto. Durante o curso, explorei como colaborar com projetos globais:

1. **Fork**: Criar uma cópia pessoal de um projeto para testar mudanças sem afetar o original.
2. **Pull Request (PR)**: Propor suas melhorias para o dono do projeto original revisar e aceitar.
3. **Issues**: Reportar bugs, sugerir novas funcionalidades ou tirar dúvidas.
4. **Licenciamento**: A importância de escolher licenças (como MIT ou Apache) para definir como outros podem usar seu código.

> "No mundo Open Source, a colaboração é mais importante que a competição."
